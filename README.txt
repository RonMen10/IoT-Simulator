
1) In order to run the simulation and obtain the results in the Output.csv file, it is required just to run the StreaminJOB.scala file.
- On the while loop it is posible to change the number of secons the symulation will run.
- Here you also find commented code for the intended streaming task.
- The Output.csv file will be saved in the "out" folder.

2) In order to run the query tasks, just run the sparkSqlQueries.java file
- On the third Query it is possible to change the queried day just by typing a different date in the variable "givenDay" the format MM/dd/YYYY.
- The results will apear in the console.
